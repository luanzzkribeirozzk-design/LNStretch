# 🔥 FF Stretch — Esticador de Tela para Free Fire

App Android com **Shizuku** — funciona em qualquer celular, **sem PC, sem root**.

## Funções

| Botão | O que faz |
|-------|-----------|
| 📐 ESTICAR | Estica a tela com % escolhida (10% a 40%) |
| ↩️ RESTAURAR | Volta resolução original |
| ⚡ OTIMIZAR FF | Libera RAM e limpa cache do Free Fire |
| 🎮 ABRIR FREE FIRE | Abre o jogo direto |

## Slider de Largura

Arraste o slider para escolher de **+10% até +40%** de largura.
O label mostra em tempo real: ex. `+15%   1080px → 1242px`
Recomendado: 15–25% (mais de 30% pode distorcer muito)

## Como Instalar

1. Vá em **Actions** no GitHub → último build → baixe **FFStretch-APK**
2. Instale no celular (ative "fontes desconhecidas")

## Configurar Shizuku (obrigatório, sem PC)

1. Instale **Shizuku** na Play Store
2. Abra o Shizuku → ative via "Depuração sem fio"
3. Abra FF Stretch → clique **AUTORIZAR**
4. Pronto!

## Fluxo de Uso

1. Abra FF Stretch → Shizuku = 🟢 Autorizado
2. Escolha a largura no slider (ex: 20%)
3. Clique **ESTICAR**
4. Clique **ABRIR FREE FIRE**
5. Ao terminar: clique **RESTAURAR**

## Compatibilidade

- Android 8+ / Todos os celulares
- Free Fire e Free Fire MAX
- Sem root, sem PC
